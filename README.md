This Raycasting software can render 3d objects such as spheres on a plane in one of about 12 different materials.

Metallic

Emissive

Lambertian:

  -Red
  -Green
  -Blue

Dielectric (glass):

  -Red
  -Green
  -Blue
  -Red and Blue
  -Green and Blue
  -Red and Green
  -Any colour

  It uses 150 ray samples per pixel that bounce a max of 20 times each which renders the objects in high definition.
